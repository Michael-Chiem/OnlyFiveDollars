This is a gift from Winged Coders. Challenge 13 might seem daunting, so I've compiled some basic instructions to help you get started. Happy Coding!

1)	Create a new repository and clone it.
2)	Copy the Challenge 13 "Develop" folder into the new repository.
3)	Right-click on server.js, open it in the integrated terminal, and run npm i. Then, exit.
4)	Rename .env.EXAMPLE to .env and add your username and password.
5)	Edit connection.js in the config folder.
6)	Modify Category.js in the models folder.
7)	Edit category-routes.js in routes > api folder.
8)	Update your server.js
9)	Right-click on schema.sql, open it in the integrated terminal.
10)	Run mysql -u root -p, enter your password, and execute "source schema.sql".
11)	Right-click on server.js, open it in the integrated terminal and run node server.js.
12)	Navigate to routes > api folder, right-click index.js, and select open in the integrated terminal. Then, run node index.js.
13)	Open Insomnia program, select GET, and enter localhost:3001/api/categories. You should see a list of products.
